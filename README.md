# Mi Proyecto de APIs Web

## Descripción
Proyecto que integra APIs de YouTube y Noticias para mostrar contenido reciente.
Aplicación web que integra APIs de YouTube y Noticias recientes, permitiendo a los usuarios buscar contenido de interés y mantenerse actualizados de manera sencilla y visual.

Descripción Larga / Funcionalidad:
Esta aplicación permite a los usuarios:

Buscar videos en YouTube según un tema específico, mostrando los resultados directamente en la página con título, canal, miniatura y enlace para verlos completos.

Consultar noticias recientes sobre un tema de interés, utilizando la API de NewsDataHub. Muestra el título, fuente, fecha, descripción y enlace a la noticia completa.

Todo se presenta en una interfaz web intuitiva y visual, con diseño moderno usando Bootstrap, lo que permite acceder a la información de forma rápida y clara.
